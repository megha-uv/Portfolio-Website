COVID-19 SYMPTOM EXTRACTION AND EVALUATION FROM SOCIAL MEDIA USING DEEP LANGUAGE MODEL

A deep language model was developed using two unique architectures, BERT and MLP, for Named Entity Recognition (NER) to identify symptom words using the BIOE tags:

  • B: Beginning of Symptom
  • I: Intermediate words of Symptom
  • O: Non-Symptom Words
  • E: End of Symptom Words

The deep language model is trained using clinical text, including personal details of patients, symptoms experienced, medications provided, and other lab results. Both BERT and MLP generate BIOE tags for each word of the input text, and their outputs are combined using an integration algorithm.


The integration algorithm prioritizes tags based on their occurrence and importance, with B tag having the highest priority. Based on the final output tags, symptom words are extracted, and a new dataset is created by labeling social media data with severity as specified by the CDC.


Machine learning classifiers, including Decision Tree Classifier, Ada Boost Classifier, SVM, and Gradient Boost Classifier, were employed for accurate severity classification.
